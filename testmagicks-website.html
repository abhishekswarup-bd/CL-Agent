Frequently Asked Questions

Everything you need to know about TestMagicks

What exams are supported? +
We support 25+ Indian competitive exams including CAT, CLAT, AILET, SNAP, NMAT, IPMAT, CUET, JEE, NEET, UPSC, and many more. We regularly add new exams based on user requests.
Can I use TestMagicks with my LLM provider? +
Yes! TestMagicks works with Claude, Gemini, Groq, and any provider through OpenRouter. You can use your own API keys or let us manage the integration. Switch between providers anytime.
How does the QC Coach Agent work? +
Every mock generated is automatically reviewed by our QC Agent. It checks for difficulty compliance, question quality, answer accuracy, and formatting. If issues are found, it regenerates that section. You get only the best content.
Is white-label available on lower tiers? +
White-label branding is included in the Institute plan (₹7,999/month). For lower tiers, you can request a custom quote if you'd like this feature. Contact our sales team to discuss options.
What's included in API access? +
API access (Institute plan only) lets you integrate TestMagicks directly into your own platform. You can automate mock generation, manage users, and sync data. Full documentation and 24/7 developer support included.
Do you offer discounts for annual plans? +
Yes! Annual plans come with 2 months free. For example, Pro plan is ₹24,999/year instead of ₹29,988. Contact sales@testmagicks.io for custom enterprise pricing and volume discounts.
